# singleTon
Atividade escolar utilizando padrão de projeto singleton.

Como exemplo, utilizei uma classe que representa a assinatura de um usuário para utilizar um determinado sistema. 
A assinatura de um usuário possui dois atributos:
status - contém o status atual da assinatura, sendo ativado ou desativado;
expiracao - contém a quantidade de dias restantes para a expiração da assinatura.
